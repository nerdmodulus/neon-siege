# 🎮 NEON SIEGE — Browser FPS

Un shooter en primera persona construido con Three.js. Corre directo en el navegador, sin instalación.

## 🕹️ Controles

| Tecla | Acción |
|-------|--------|
| `WASD` / `↑↓←→` | Mover |
| `Mouse` | Apuntar |
| `Click izquierdo` | Disparar |
| `R` | Recargar |
| `Shift` | Correr |
| `Esc` | Pausar / liberar mouse |

## 🚀 Jugar online (GitHub Pages)

1. Sube este repo a GitHub
2. Ve a **Settings → Pages**
3. En *Source*, selecciona `main` / `root`
4. Guarda y espera ~1 minuto
5. Abre la URL que te da GitHub Pages

## 🎯 Mecánicas

- **Oleadas progresivas** — cada wave trae más enemigos y más difíciles
- **IA enemiga** — los enemigos te persiguen y disparan
- **Sistema de armas** — recarga, munición limitada
- **Minimapa** — orientación táctica en tiempo real
- **Sistema de vida** — daño por bala y por contacto

## 🛠️ Tecnología

- **Three.js r128** — motor 3D WebGL
- **Raycasting** — detección de impactos instantánea
- **Pathfinding básico** — enemigos con navegación por grid
- HTML/CSS/JS puro — sin frameworks, sin build tools
